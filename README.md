# Justin Bull - Portfolio Website

Personal portfolio and resume website showcasing my work experience, projects, and creative side projects.

## About

This is my personal portfolio site built with vanilla HTML/CSS/JavaScript. It features:

- Work experience timeline
- Project showcase with live demos
- Photo gallery slider
- Video portfolio with YouTube integration
- Interactive particle background using Three.js

## Tech Stack

- **Frontend**: Vanilla HTML5, CSS3, JavaScript (ES6)
- **Libraries**:
    - Three.js (particle effects)
    - Snap Touch (photo slider)
    - AOS (scroll animations)
    - YouTube IFrame API (video player)

## Local Development

No build process required. Just open `index.html` in your browser or use a simple HTTP server:

```bash
# Python
python -m http.server 8000

# PHP
php -S localhost:8000

# Node
npx http-server
```

Then visit `http://localhost:8000`

## Features

- Responsive design
- Lazy loading images
- Scroll-based particle animation
- Interactive video modal
- Touch-enabled photo slider
- Smooth scroll animations
