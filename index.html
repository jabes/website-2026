What I've Built

Projects built to solve problems, learn something new, or just because.

Pinup

Pinup.js is a lightweight, standalone JavaScript library for tagging content within images. Click anywhere on an image to add titles, keywords, and URLs. Simple and accessible for all developers, it offers extensive customization options, features, and themes for those who need more control.

Tofu

A tiny, efficient JavaScript library developed in 2010 that focuses on core element manipulation, including events, animations, DOM traversal, and HTML parsing. Lightweight and performant, it provides essential utilities for common web development tasks without the overhead of larger frameworks.

Terrace Game

A grid-based side-scrolling game built from scratch using the Processing library. Sprites are borrowed from classic NES titles including Mega Man, Mario, and Sonic The Hedgehog. This non-commercial project serves as a demonstration of Java programming abilities and game development skills.

Terrace Map Editor

This is the map editor program for the game Terrace. Create and modify custom map files with ease using an intuitive drag-and-drop interface. Design unique terrain, place objects, and customize elements. Once saved, load your custom maps into the game via the in-game menu and enjoy playing them.

Snap Touch

A touch-enabled, responsive slider for smooth horizontal navigation on mobile and desktop devices. Built with native JavaScript (ES6) without libraries, it intelligently calculates velocity to create natural deceleration when flicked and automatically snaps to the nearest element when motion finishes.

Pokédex

A React app for building a Pokémon team. Search with typeahead suggestions grouped by type. Add up to six Pokémon, view their stats (HP, speed, defense, attack, special stats), sprites, and remove them as needed. Search field disables at full team capacity. Zero state displays when empty.
